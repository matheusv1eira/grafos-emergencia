# Experimento Científico em Grafos - Busca em Grafos Planos para Resposta a Emergências

## Descrição
Este projeto compara algoritmos de busca (BFS, Dijkstra e A*) em redes viárias urbanas para aplicações de resposta a emergências.

## Estrutura
- : Dados das malhas viárias
- : Jupyter notebooks para análise
- : Código fonte em Python
- : Resultados dos experimentos (CSV, gráficos)
- : Testes unitários
